## 1. CONSENTIMENTO PARA ADMINISTRADORES DE BARBEARIAS

Ao criar uma conta como **administrador** de barbearia no Shafar, você consente expressamente que o Shafar colete, armazene, utilize e processe os seguintes dados pessoais:

### 1.1. Dados Pessoais Coletados

**a) Dados de Identificação:**
- [ ] Nome completo
- [ ] E-mail de login
- [ ] Senha (criptografada)

**b) Dados da Barbearia:**
- [ ] Nome da barbearia
- [ ] Endereço comercial completo
- [ ] Telefone de contato
- [ ] E-mail comercial
- [ ] Logo/imagem (opcional)
- [ ] CNPJ ou CPF (MEI)

**c) Dados de Pagamento:**
- [ ] Informações de faturamento
- [ ] Dados de cartão de crédito (processados pelo Stripe, não armazenados por nós)
- [ ] Histórico de transações

**d) Dados de Uso:**
- [ ] Endereço IP
- [ ] Logs de acesso
- [ ] Horários de utilização
- [ ] Páginas visitadas
- [ ] Dispositivo e navegador

### 1.2. Finalidades do Tratamento

Seus dados serão tratados para as seguintes finalidades:

- [ ] **Prestação do serviço**: Permitir acesso à Plataforma, gerenciar agendamentos, profissionais e clientes
- [ ] **Autenticação**: Validar login e manter sessão segura
- [ ] **Cobrança**: Processar pagamentos e emitir faturas
- [ ] **Comunicação**: Enviar notificações sobre agendamentos, atualizações e suporte
- [ ] **Melhorias**: Analisar uso para aprimorar funcionalidades
- [ ] **Segurança**: Prevenir fraudes, abusos e proteger a plataforma
- [ ] **Compliance**: Cumprir obrigações legais e regulatórias

### 1.3. Compartilhamento de Dados

Seus dados poderão ser compartilhados com:

- [ ] **Supabase Inc.** (hospedagem de banco de dados - EUA)
- [ ] **Vercel Inc.** (hospedagem da aplicação - EUA)
- [ ] **Stripe Inc.** (processamento de pagamentos - EUA)
- [ ] **SendGrid/Twilio** (envio de e-mails - EUA)
- [ ] **Sentry** (monitoramento de erros - EUA)
- [ ] **Autoridades competentes** (mediante ordem judicial ou requisição legal)

### 1.4. Transferência Internacional

[ ] Consinto expressamente que meus dados sejam transferidos e armazenados em servidores localizados nos **Estados Unidos** e outros países, mediante garantias de segurança e cláusulas contratuais adequadas.

### 1.5. Prazo de Retenção

Seus dados serão mantidos:

- **Enquanto sua conta estiver ativa**: Para prestação do serviço
- **Até 6 meses após inatividade**: Para possível reativação
- **Até 5 anos**: Dados de pagamento (obrigação legal - fins fiscais)
- **Indefinidamente**: Dados anonimizados ou agregados (para estatísticas)

### 1.6. Direitos do Titular

Você tem os seguintes direitos garantidos pela LGPD:

- [ ] **Confirmar** a existência de tratamento de seus dados
- [ ] **Acessar** seus dados pessoais
- [ ] **Corrigir** dados incompletos, inexatos ou desatualizados
- [ ] **Anonimizar, bloquear ou eliminar** dados desnecessários
- [ ] **Solicitar portabilidade** de dados a outro fornecedor
- [ ] **Eliminar dados** tratados com base em consentimento
- [ ] **Obter informações** sobre compartilhamento de dados
- [ ] **Revogar consentimento** a qualquer momento

**Para exercer seus direitos**: privacy@shafar.com.br ou dpo@shafar.com.br

### 1.7. Revogação de Consentimento

[ ] Estou ciente de que posso **revogar este consentimento a qualquer momento**, mediante solicitação via e-mail para privacy@shafar.com.br.

**Consequências da revogação**:
- Impossibilidade de continuar utilizando a Plataforma
- Cancelamento da conta
- Exclusão dos dados pessoais (ressalvadas obrigações legais de retenção)
